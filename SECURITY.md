# Security Policy

## Supported Scope
Fokus keamanan saat ini:
- API authentication dan authorization
- Proses provisioning sistem Linux
- Penyimpanan state dan backup

## Reporting a Vulnerability
- Jangan buat issue publik untuk kerentanan.
- Kirim laporan private melalui GitHub Security Advisories:
  - `https://github.com/wayangm/Nusantara-Panel/security/advisories/new`
- Sertakan:
  - deskripsi dampak,
  - langkah reproduksi,
  - versi/commit terdampak,
  - saran mitigasi jika ada.

## Response Target
- Triase awal: maksimal 3 hari kerja.
- Patch awal untuk high severity: target 7 hari kerja.
