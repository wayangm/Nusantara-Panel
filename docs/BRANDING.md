# Branding Guide - Nusantara Panel

## Nama Produk
- Primary: `Nusantara Panel`
- Binary daemon: `nusantarad`
- Systemd service: `nusantara-panel`

## Tagline
- `Kelola server Linux lebih cepat, aman, dan terarah.`

## Gaya Penulisan
- Gunakan `Nusantara Panel` untuk nama produk di dokumen.
- Gunakan huruf kecil untuk nama teknis (`nusantarad`, `nusantara-panel`, `NUSANTARA_*`).
- Hindari referensi nama produk lain di dokumen resmi.

## Namespace Teknis
- Environment variables: `NUSANTARA_*`
- Direktori default:
  - `/etc/nusantara-panel`
  - `/var/lib/nusantara-panel`
  - `/var/log/nusantara-panel`
  - `/var/backups/nusantara-panel`
