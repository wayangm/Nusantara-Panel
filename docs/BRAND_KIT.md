# Brand Kit - Nusantara Panel

## Brand Core
- Product name: `Nusantara Panel`
- Tagline: `Kelola server Linux lebih cepat, aman, dan terarah.`
- Brand attributes: cepat, tegas, stabil, operasional.

## Naming Rules
- Use `Nusantara Panel` in product copy.
- Use `nusantarad` for daemon/binary.
- Use `nusantara-panel` for systemd service/package naming.
- Use `NUSANTARA_*` for environment variables.

## Visual Direction
- Primary color: `#0D9488` (teal)
- Secondary color: `#0369A1` (blue)
- Accent color: `#34D399` (mint)
- Neutral dark: `#0F172A`
- Neutral light: `#F8FAFC`

## Typography Direction
- Heading: `Plus Jakarta Sans` (600/700)
- Body: `Source Sans 3` (400/500)
- Monospace: `JetBrains Mono`

## Logo Assets
- Primary mark (SVG): `assets/brand/nusantara-panel-logo.svg`

## Copy Tone
- Use direct operational language.
- Prioritize command clarity over marketing claims.
- Avoid references to competitor products in official copy.
